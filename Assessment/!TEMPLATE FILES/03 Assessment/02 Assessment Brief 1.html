Coursework Overview

Module: Introduction to Computer Programming (SOFT40161)

Total Marks: 100

Submission Deadline: See the Coursework 

Submission Format: A single Jupyter Notebook and GitHub repository link

Coursework Description

In this coursework, students will apply their programming knowledge to perform data analysis on a real-world dataset. The task will encompass key aspects of Python programming, such as control structures, functions, data handling, data visualization, and graphical user interface (GUI) development. Additionally, students are required to use version control and provide detailed documentation and a report summarizing their work.

The coursework is divided into six sections, each focusing on specific programming and data science concepts:

Sections Breakdown

Section 1: Control Structures (16 marks)

  • Task: Implement control structures (e.g., if-statements, loops) to analyze a dataset.
  • Dataset: Select a publicly available dataset (e.g., from Kaggle, UCI Machine Learning Repository).
  • Objective: Write Python code that uses control structures to identify trends, averages, or anomalies based on specific conditions.
  • Skills: Basic Python programming, control structures, data analysis.

Section 2: Functions and Modules (16 marks)

  • Task: Design and implement functions and use external modules to handle data analysis.
  • Objective: Create modular functions for different parts of the analysis and use at least 3 external modules (e.g., NumPy, Requests) for enhanced functionality.
  • Skills: Writing reusable functions, importing and using Python modules.

Section 3: Data Handling with Pandas (16 marks)

  • Task: Perform data manipulation and analysis using Pandas.
  • Objective: Clean the dataset (handling missing values, duplicates), filter, and aggregate data for deeper analysis. Document steps and explain the logic behind the methodology.
  • Skills: Data cleaning, filtering, aggregating, and documenting Python code.

Section 4: Data Visualization (16 marks)

  • Task: Create visual representations of the data using Python libraries like Matplotlib, Seaborn, or Plotly.
  • Objective: Generate at least three types of visualizations with proper customizations (subplots, interactive elements).
  • Skills: Data visualization, Python plotting libraries, customizing plots for effective data communication.

Section 5: GUI Development (16 marks)

  • Task: Develop a simple graphical user interface (GUI) using Tkinter or PyQt to interact with the dataset.
  • Objective: The GUI should allow users to interact with the data, such as selecting subsets, displaying summary statistics, and generating visualizations through user input.
  • Skills: GUI development, Python interface design, usability focus.

Section 6: Version Control, Documentation, and Report (20 marks)

  • Version Control: Set up and maintain a GitHub repository for the project, committing changes incrementally with clear commit messages.
  • Report: Document the entire project within the Jupyter Notebook, including:
    • Introduction: Overview of the dataset and problem.
    • Methodology: Description of data processing and analysis techniques.
    • Findings: Insights from data analysis and visualizations.
    • Conclusion: Summary of results and implications.
  • Code Documentation: Thoroughly document the code with comments and docstrings for clarity.
  • Skills: GitHub for version control, technical writing, documenting code.

Submission Requirements

  1. Jupyter Notebook: A well-documented notebook containing all code, visualizations, and explanations.
  2. GitHub Repository: A link to a GitHub repository that shows incremental commits, project structure, and version control usage.
