Module Overview 

Module aims

This Module Aims:

  • To cultivate proficiency in efficiently developing data-driven solutions for real-world, practical problems using Python programming.
  • To foster competence in using integrated development environments (IDEs) and data science tools to streamline the programming workflow.
  • To build advanced skills in procedural programming and data manipulation, enhancing problem-solving and logical thinking in the context of data science.
  • To introduce and strengthen familiarity with object-oriented programming (OOP) principles and Python libraries for AI and data science, enabling the design of scalable, reusable, and maintainable code for machine learning and data analytics.

Delivery mode

The module will be delivered through a combination of in-person 2-hour lectures, providing theoretical foundations and demonstrations, and 2-hour lab sessions, offering hands-on programming practice and problem-solving exercises. Additionally, 1-hour optional Surgery sessions will be available weekly, allowing students to seek clarification, receive guidance on specific challenges, and enhance their understanding of the material. This balanced approach ensures both theoretical knowledge and practical application, with ample support for individual learning needs..

Module content

The SOFT40161: Introduction to Computer Programming module provides a comprehensive introduction to programming using Python, with a specific focus on its applications in data science and artificial intelligence (AI). This 10-week module is designed for students from diverse academic backgrounds, offering a blend of theoretical concepts and practical experience to develop essential programming skills.

The module will cover:

  • Python Environment Setup, Data Types, and Operations: Students will begin by setting up the Python programming environment, learning basic syntax, and understanding data types (integers, strings, floats) and operations.
  • Data Structures and Control Structures: Students will explore core data structures (lists, tuples, sets, dictionaries) and implement control structures (loops, conditionals) for decision-making and flow control in programs.
  • Procedural Programming: Students will dive into functions, including parameter passing, recursion, and handling exceptions, building modular, efficient, and reusable code.
  • Object-Oriented Programming (OOP): Key OOP concepts like classes, objects, methods, inheritance, and constructors will be introduced to help students design structured, maintainable programs.
  • Working with Data: Students will learn to manage data through file I/O operations and handle large datasets efficiently using the powerful NumPy library for numerical computations.
  • Data Manipulation and Analysis: The module introduces Pandas, a widely-used library for data manipulation, allowing students to clean, process, and analyze datasets efficiently.
  • Data Visualization: Students will create informative visual representations of data using Matplotlib, from basic plots to advanced customizations.
  • Machine Learning Fundamentals: The basics of machine learning in Python will be introduced, with students working on real-world datasets to build and evaluate simple models.
  • GUI Development: Students will also learn to develop graphical user interfaces (GUIs) with Python, bringing interactivity to their applications.

The module’s structure combines 2-hour weekly lectures, 2-hour lab sessions for hands-on practice, and optional 1-hour Surgery sessions to provide additional support. By the end of the module, students will have developed the foundational skills necessary to apply Python programming in real-world scenarios related to data science and AI.

Peer support and collaboration

Although the coursework assessment will be conducted individually, students are encouraged to collaborate and engage through various channels. During lab sessions and optional Surgery hours, students can discuss ideas, troubleshoot code, and share resources. Additionally, online discussion boards and group chats will be available for ongoing peer support. While direct collaboration on assignments is not allowed, these platforms will foster a collaborative learning environment where students can support each other’s understanding and growth.
